FAQ

      

        

          
Why can I not add a remote?

          
The current folder must already be a Git repository. If it is not, click Init first or choose the correct project root.

        

        

          
Why does commit fail?

          
Usually one of these is missing: staged files, user.name, or user.email. Configure Git identity once in your terminal if needed.

        

        

          
Why does push fail?

          
Push depends on remote configuration, branch selection, and Git authentication. Private repositories are allowed, but you still need valid credentials.

        

        

          
Why is the page slow with large projects?

          
Very large working trees can create hundreds or thousands of file changes. The app now limits the first render to keep the UI responsive. Add a proper .gitignore so build output and dependency folders do not flood the status list.

Best practice

          
Do not point the app at your whole drive or at folders full of generated files. Use the actual project root and keep generated output ignored.
